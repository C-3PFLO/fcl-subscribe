subscribe(options) → {function}

Subscribe to an @onflow/fcl request across a range of block heights, including newly created blocks.
Parameters:
Name Type Description
options Object
Properties
Name Type Attributes Default Description
block function fcl.block function, used to determing current block height
getQuery function an fcl.send function, called with the current block context to query the next range
onResponse function response handler, which should call fcl.decode to parse the response data. If the response handler is asynchronous it can return a promise to wait for the handle to complete.
onError function <optional>
 error handler
range Integer <optional>
 249 block range to query per iteration
fromBlockHeight Integer <optional>
 <current> starting block height
sleepTime Integer <optional>
 60000 time to sleep between iterations when remaining blocks <= 250 (ie: reading from blocks as they are finalized). When remaining blocks > 250, sleep time is 1000ms.
abortOnError Boolean <optional>
 false abort subscription on error
